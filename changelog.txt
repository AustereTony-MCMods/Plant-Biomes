1.4.0
+Added settings overlay for supported plants. Overlay can be enabled with "/pb enable-overlay"
and disabled with "/pb disable-overlay". Overlay radius is 20 blocks.
+Added command "/pb reload" for forced settings reload.
+Fixed carrot and potato crops identification (added to supported plants filter).
+Added support for HarvestCraft crops, saplings and fruits. 
+Added support for AgriCraft crops.
1.3.12
+Code improvements. Coremod rewritten and signed. 
+Improved plants identification. Unsupported plants will not be 
identified now.
+Added support for vanilla mushrooms.
+Added support for vanilla nether wart identify.
+Added support for Biomes O' Plenty grass block.
+Added support for Biomes O' Plenty bamboo.
1.3.6
+Configs update algorithms optimization and fixes
for internal configuration files (if enabled).
+Added support for vanilla reed.
+Added support for vanilla cactus.
+Added support for Forestry saplings.
1.3.2
+Added way to check biome for ic2 crops. Bonemeal 
usage will reveal denied biome with gray particles.
+Performance optimizations.
1.3.0
+Settings of any subtype can be marked as global and 
will be applied for every subtype (e.g all crop stages will use
settings from marked subtype).
+Added support for vanilla cocoa beans.
+Added support for ic2 crops.
1.2.0
+Added "autosave" flag to config ("true" by default). 
If true, any changes (with command "/pb") will be saved automatically.
+Settings status (disabled or enabled) now will be saved 
and applied after game restart.
+Added ability to create biome whitelists for plants.
Plants will only grow in whitelisted biomes, any other
settings will be ignored (if whitelist isn't empty).
+Added support for vanilla vines.
+Improved plants display names used in chat logs.
If block hasn't name (e.g. crops and saplings stage),
its name will be displayed as "Undefined" instead "Air".

+Added support for Twilight Forest saplings
+Added support for Biomes O' Plenty saplings
